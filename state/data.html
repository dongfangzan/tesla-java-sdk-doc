# 获取车辆详情

# GET /api/1/vehicles/{id}/vehicle_data

TIP

这个接口需要车辆处于唤醒状态

# Url Parameters

参数 例子 描述
endpoints charge_state;climate_state 指定返回你需要的状态数据，多个用;分割
